.04 — About

Problem
→ Solution

A hackathon project solving one of the most overlooked everyday frustrations — finding parking and finding your car after.

01
The Problem

Large parking areas in malls, hospitals, and airports cost people 10–20 minutes every visit.

  • No visibility of empty slots
  • Circling causes congestion
  • Forgetting car location post-shopping
  • Stressful for elderly & disabled
  • Zero real-time data
02
Our Solution

ParkEase gives users a real-time visual map, one-click booking, and GPS-style car navigation.

  • Visual 100-slot map with colors
  • Instant booking with details form
  • Car finder with step-by-step route
  • QR scan for quick slot detection
  • Admin dashboard for oversight
03
Tech Stack

Zero dependencies. Runs anywhere. Built to demo and built to ship.

  • Pure HTML + CSS + JavaScript
  • localStorage persistence
  • Canvas API route animation
  • Mobile-first responsive layout
  • No backend required
